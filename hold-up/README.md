# HOLD UP

**The private assistant between persuasion and action.**

HOLD UP turns screenshots, shared messages, links, bills, receipts, and optional Gmail data into clear warnings, reminders, recurring obligations, subscription controls, calendar actions, and safe next steps.

## Product promise

- Screenshot it or share it to HOLD UP.
- Understand what it is asking you to do.
- Surface money, risk, dates, renewals, and cancellation details.
- Turn the result into an action card instead of a long AI answer.
- Keep sensitive data private and user-controlled.

## Core experiences

1. **Analyze shared content** — screenshots, images, links, text, and PDFs.
2. **Create useful action cards** — Verify Safely, Add to Calendar, Set Reminders, Mark Paid, Cancel Subscription, Save Evidence, Contact Trusted Person, or Start Recovery.
3. **Remember recurring bills** — merchant, amount history, due day, preferred pay day, reminder offsets, autopay state, and future monthly occurrences.
4. **Find subscriptions** — optional Gmail analysis with explicit consent, cost estimates, billing cadence, renewal dates, annual totals, and cancellation routes.
5. **Protect privacy** — local processing where practical, minimal retention, visible permissions, redaction, and deletion controls.

## Repository layout

- `web/` — mobile-first product prototype and GitHub Pages test surface.
- `android/` — native Android implementation plan and future application source.
- `docs/` — product requirements, architecture, privacy, and research.

The existing Project Midnight game remains at the repository root until HOLD UP is clearly more useful, tested, and ready to replace it.
